Web Calculator
This is a simple web-based calculator built using HTML, CSS, and JavaScript. It provides a clean and user-friendly interface to perform basic arithmetic operations.

Features
Addition
Subtraction
Multiplication
Division
Responsive design
Clear and user-friendly interface
